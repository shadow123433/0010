Política de Privacidade

Esta plataforma respeita a privacidade dos usuários e protege seus dados pessoais conforme a Lei Geral de Proteção de Dados (LGPD).

1. Dados coletados

Podemos coletar informações como nome, escola, produtos selecionados e dados de contato.

2. Finalidade

Os dados são utilizados exclusivamente para processamento dos pedidos e comunicação com a loja responsável.

3. Compartilhamento

As informações podem ser compartilhadas apenas com a loja parceira para atendimento do pedido.

4. Armazenamento e segurança

Os dados são armazenados em banco de dados protegido e acessados apenas por pessoas autorizadas.

5. Direitos do usuário

O usuário pode solicitar correção ou exclusão de seus dados.

6. Contato

Para dúvidas sobre privacidade, entre em contato com a loja responsável.
